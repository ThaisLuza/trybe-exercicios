Top 3 dos melhores sites que existem na web

  1. YouTube
  2. Netflix
  3. Duolingo

Principais motivos para eu gostar deles

  • No YouTube é possível aprender qualquer coisa, fazer cursos, conhecer várias culturas diferentes, procurar por músicas que nem mesmo sabemos o nome direito, além de assistir indianos construindo piscinas.
  • A Netflix é uma serviço de transmissão online que oferece uma ampla variedade de séries, filmes e documentários. É incrível quanto tempo passamos escolhendo um filme ou série que não vamos gostar.
  • O Duolingo é maravilhoso, a gente aprende além do verbo to be, e todo dia tem 259 notificações para você não esquecer de praticar.
